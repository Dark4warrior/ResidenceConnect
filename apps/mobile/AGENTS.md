# Expo SDK 54

Ce projet utilise **Expo SDK 54** (compatible avec l'app Expo Go publiée sur l'App Store / Play Store).

Consulte la documentation versionnée exacte avant d'écrire du code :
https://docs.expo.dev/versions/v54.0.0/

Ne pas remonter au SDK 56+ : l'Expo Go public ne le supporte pas encore (TestFlight uniquement).
